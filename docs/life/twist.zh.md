# 手腕不适尺侧腕屈肌 与 肱桡肌

## 1. 尺侧腕屈肌 (FCU)
**定位**：前臂内侧（小拇指侧），最肉的地方。
**别名**：“键盘手”的受害者。

### 🚨 为什么疼？（病因分析）
* **核心原因**：HHKB 的 `Fn` 组合键操作。
* **动作机制**：
    * 为了用小拇指去够右下角的键，手腕被迫做**大幅度向外扭（尺偏）** + **用力下压**。
    * 这是 FCU 的主要职责，高频重复导致其过劳并形成扳机点。

### 🛠️ 修复方案

#### A. 筋膜球/按摩球放松
* **位置**：前臂内侧最酸痛的肌肉肚。
* **操作**：利用手臂自重按压并停留，配合深呼吸，直到肌肉松开。不要快速来回滚，要深压。

#### B. 精准拉伸：反向伸腕 (加强版)
* **核心逻辑**：对抗“屈腕+向外偏”的僵局，反向做“伸腕+向内偏”。
* **步骤分解**：
    1.  **锁肘**：右手向前伸直，**手肘必须完全伸直**（关键，否则卸力）。
    2.  **手势**：掌心朝前，指尖向下（像“停止”手势倒过来）。
    3.  **抓握**：左手握住右手四指，发力点侧重在**小拇指/无名指**根部。
    4.  **拉伸**：左手将右手掌向身体方向拉（背屈）。
    5.  **旋转 (灵魂动作)**：在拉的同时，让右手产生微小旋转，即**把小拇指往右耳垂方向掰**（桡偏）。
    6.  **保持**：15-30秒，感受前臂内侧有一条筋被强力拉开。

## 2. 肱桡肌 (Brachioradialis)
**定位**：前臂外侧上方，连接手腕与上臂（做“点赞”姿势时前臂鼓起的那块肉）。
**别名**：“鼠标手”与“悬腕”的受害者。

### 🚨 为什么疼？（病因分析）
* **核心原因**：悬空打字（无手托） + 鼠标抓握。
* **动作机制**：
    1.  **静力支撑**：当肘部无支撑时，它负责持续“吊着”前臂（屈肘）。
    2.  **旋前姿态**：掌心向下（打字/握 MX Master）时，肌肉被拉长且持续紧张，久之变成硬结。

### 🛠️ 修复方案

#### A. 特效按摩：按压 + 转动 (Pin & Stretch)
* **原理**：单纯拉伸很难拉到深层，需要物理剥离。
* **步骤分解**：
    1.  **定位**：屈肘90度（大拇指朝上），找到肘窝外侧最厚实、按下去最酸爽的硬结。
    2.  **按死**：左手拇指狠狠按住痛点，**不要滑动**，保持压力。
    3.  **转动**：在按住不动的状态下，右手慢慢做**“旋转门把手”**动作（翻掌心：上 -> 下 -> 上）。
    4.  **感受**：你会感觉到肌肉在拇指下被强行剥离、滑动。极痛但极解压。
    5.  **频率**：每组转动 10-15 次。

#### B. 针对拉伸：握拳下压
* **步骤分解**：
    1.  **藏指**：右手握拳，把**大拇指包裹在拳心内**。
    2.  **内旋**：整条手臂向内旋转，使手背对着身体内侧。
    3.  **下压**：用左手压住右拳的拳面，向下方压手腕。
    4.  **感受**：明显的拉扯感从大拇指根部一直延伸到手肘外侧上方。

## 3. 环境与习惯微调 (根治策略)

* **🚫 拒绝 `Fn` + `,`**：这是 FCU 疼痛的根源。请改为 `HJKL` 映射或使用 HHKB Studio 左侧触摸条。
* **💺 寻找肘部支撑**：调高椅子扶手，托住手肘，让肱桡肌“下班”，不再负责吊着前臂。
* **🖱️ 放松鼠标抓握**：手掌轻搭在鼠标上，不要用大鱼际死扣。
* **🩹 维持中立位**：利用手托或拳击绷带，强制手腕保持平直，避免过度的背屈。
